**Дата:** 2026-03-16
**Референс:** `/Users/ivan/Downloads/wd_example.jpg`

## Цель

Переработать визуальный стиль свадебного лендинга по образцу из референса: шалфейно-зелёные секции, арочные переходы, белые сердечки-разделители, крупные uppercase-заголовки, pill-таймер.

## Цветовая палитра (полная замена)

| CSS-переменная | Значение | Назначение |
|---|---|---|
| `--sage` | `#4a7a78` | Основной зелёный (фон секций) |
| `--sage-dark` | `#3a6360` | Тёмно-зелёный (hover, акценты) |
| `--sage-light` | `#6a9e9b` | Светло-зелёный (вторичные акценты) |
| `--white` | `#ffffff` | Белый (фон чередующихся секций) |
| `--off-white` | `#f5f5f3` | Слегка тёплый белый (hero, формы) |
| `--text` | `#2c2c2c` | Основной тёмный текст |
| `--text-muted` | `#666660` | Приглушённый текст |
| `--text-on-sage` | `#ffffff` | Текст на зелёном фоне |

Удалить: `--cream`, `--ivory`, `--gold`, `--gold-light`, `--gold-dark`, `--brown`, `--dark`, `--text-light`.

## Арочные переходы

Каждая зелёная секция имеет арку снизу — белый полукруг, перекрывающий низ зелёного блока.
Реализация: `::after` псевдоэлемент с `background` следующей секции + `border-radius: 50%`.

```css
.section--sage::after {
  content: '';
  position: absolute;
  bottom: -1px;
  left: -5%;
  width: 110%;
  height: 60px;
  background: /* цвет следующей секции */;
  border-radius: 50% 50% 0 0;
}
```

## Белые сердечки-разделители

SVG сердечко (outline) белого цвета, размер ~32px, центрировано между секциями.
HTML-элемент `.section-heart` вставляется в конце каждой зелёной секции.

## Типографика

Заголовки секций переделываются в стиль референса:

- Шрифт: Montserrat (sans-serif), `font-weight: 700`
- Размер: `clamp(2.5rem, 8vw, 4.5rem)`
- `text-transform: uppercase`
- `letter-spacing: 0.2em`
- Цвет: белый на зелёном, `--sage` на белом

Русские названия секций по образцу:
- Save the Date → **«КОГДА?»** (оставить как есть, изменить стиль)
- Программа → **«ПРОГРАММА»**
- Место → **«ЛОКАЦИЯ»**
- RSVP → **«АНКЕТА ГОСТЯ»** (подзаголовок: «пожалуйста заполните анкету»)

## Обратный отсчёт

Было: крупные цифры на тёмно-коричневом фоне.
Станет: pill-бейджи (`border-radius: 999px`, зелёный фон + белый текст) на светлом фоне секции.

```
[ 86 ]   [ 23 ]   [ 45 ]   [ 12 ]
 дней     часов    минут   секунд
```

## Кнопки

- `border-radius: 999px` (полностью скруглённые)
- Фон: `--sage-dark` (на зелёных секциях — белый/прозрачный, на белых — зелёный)
- Убрать: `border: 1px solid var(--gold)`

## Навигация

- Фон: `rgba(74,122,120,0.95)` (полупрозрачный зелёный) или белый
- Ссылки: белые или тёмные

## Не меняется

- Структура HTML (секции, id, якоря)
- Весь TypeScript (countdown, gallery, RSVP, scroll-анимации)
- Календарь с пульсирующим сердечком (только обновить цвет heart SVG на белый для зелёного фона)
- Lightbox галереи
- Адаптивная вёрстка
